# Appium Selenium Kotlin Gradle Simple project
## Description
- Appium
- Selenium
- Kotlin
- Gradle
- TestNG

#### Tests:
- WEB (Selenium)
    - Google Chrome
- Mobile (Appium + Selenium)
    - Android

#### Run web tests 
```
./gradlew clean test -PwebTest
```
#### Run mobile tests
```
./gradlew clean test -PmobileTest
```
#### Run all tests
```
./gradlew clean test
```
