# Appium / Selenium Kotlin simple project
## Description
- Appium
- Selenium
- Kotlin
- Gradle
- TestNG

#### Tests:
- WEB (Selenium)
    - Google Chrome
- Mobile (Appium + Selenium)
    - Android
    - Precondition: The app is installed on the mobile device

#### Run web tests 
```
./gradlew clean test -PwebTest
```
#### Run mobile tests
```
./gradlew clean test -PmobileTest
```
#### Run all tests
```
./gradlew clean test
```
